## Electron app.
### Basic setup with React, Typescript and Webpack.

To run this project.

- Clone this repo

```bash
  git clone https://github.com/elisealcala/electron-react-ts.git
```

- Install the dependencies

```bash
  npm install
```

- Now run these two commands in separate consoles.

```bash
  npm run dev:react
```

```bash
  npm run dev:electron
```
